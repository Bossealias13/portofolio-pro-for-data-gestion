# Portfolio Professionnel : Data Analyst & Expert ADV

Bienvenue sur mon portfolio ! Ce site présente mes compétences en analyse de données (BI) et en gestion administrative (ADV). Il a été conçu pour être rapide, accessible et moderne.

## 🚀 Lien vers le site en ligne
[Cliquez ici pour voir mon portfolio en action](https://ton-pseudo.github.io/portfolio-data-gestion/)

## 🎯 À propos
Ce projet a été réalisé pour mettre en avant ma double compétence :
* **Data Analyst** : Maîtrise de la BI (Power BI, Looker Studio, Excel Avancé).
* **Expert ADV** : Maîtrise des flux de gestion (Sage 100, SYSCOHADA).

## 🛠 Stack Technique
* **Structure** : HTML5 Sémantique
* **Style** : CSS3 (Grid, Flexbox, Variables CSS, Mobile-First)
* **Hébergement** : GitHub Pages

## 📂 Organisation du code
- `/assets/css/` : Style structuré par thèmes.
- `/assets/js/` : Logique de navigation active.
- `/assets/images/` : Assets visuels optimisés.
